    Guide des licences VOÏA¶

    Comprendre vos limites et options de licence.

    Qu'est-ce qu'une licence ?¶

    Votre licence VOÏA détermine :

    • Le nombre de campagnes par an
    • Le nombre d'entreprises clientes distinctes que vous pouvez sonder (toutes campagnes confondues)
    • Le nombre d'utilisateurs de la plateforme
    • Les fonctionnalités disponibles
    • Le niveau de support

    Consulter votre licence : Paramètres → Informations sur la licence

    Niveaux de licence¶

    Core¶

    Pour : petites équipes débutant avec la collecte de feedback

    Limites : - 4 campagnes / an (limites personnalisées disponibles — voir ci-dessous) - 5 utilisateurs maximum (limites personnalisées disponibles — voir ci-dessous) - 20 entreprises clientes (toutes campagnes confondues, cumulatif) - Durée : 12 mois

    Fonctionnalités : - ✓ Sondages classiques - ✓ Sondages conversationnels - ✓ Suivi NPS - ✓ Envoi par email - ✓ Analyse de sentiment - ✓ Analyse des thèmes clés - ✓ Analytiques avancés - ✓ Génération de rapports exécutifs - ✓ Export de données (CSV / Excel) - ✓ Modèles de sondage personnalisés - ✓ Intelligence QBR - ✓ Service d'assistance (help desk)

    Plus (recommandé)¶

    Pour : entreprises en croissance

    Limites : - 4 campagnes / an (limites personnalisées disponibles — voir ci-dessous) - 10 utilisateurs maximum (limites personnalisées disponibles — voir ci-dessous) - 100 entreprises clientes (toutes campagnes confondues, cumulatif) - Durée : 12 mois

    Fonctionnalités : - ✓ Sondages classiques - ✓ Sondages conversationnels - ✓ Suivi NPS - ✓ Envoi par email - ✓ Analyse de sentiment - ✓ Analyse des thèmes clés - ✓ Analytiques avancés - ✓ Génération de rapports exécutifs - ✓ Export de données (CSV / Excel) - ✓ Modèles de sondage personnalisés - ✓ Intelligence QBR - ✓ Service d'assistance (help desk) - ✓ Accès API

    Remarque : Core et Plus incluent le même ensemble complet de fonctionnalités, à l'exception de l'accès API, disponible uniquement sur Plus et Pro. Les autres différences entre ces niveaux concernent uniquement les limites d'utilisation (utilisateurs, entreprises clientes, campagnes).

    Limites personnalisées pour Core et Plus¶

    Les administrateurs de plateforme peuvent configurer des limites d'utilisateurs et de campagnes personnalisées pour les licences Core et Plus — par exemple, augmenter le quota de campagnes ou le nombre d'utilisateurs autorisés au-delà des valeurs standard. Ces limites personnalisées sont accordées par un administrateur de plateforme et ne nécessitent pas de passage à Pro. Contactez votre administrateur de compte ou les ventes VOÏA pour en faire la demande.

    Pro¶

    Pour : entreprises à grande échelle

    Limites (personnalisables) : - 12+ campagnes / an (personnalisable) - 25+ utilisateurs (personnalisable) - Entreprises clientes illimitées (configurable par l'administrateur plateforme) - Durée : 12+ mois (personnalisable)

    Fonctionnalités : - ✓ Tout ce qui est inclus dans Plus - ✓ Branding white-label - ✓ Intégrations personnalisées - ✓ Garantie SLA (disponibilité 99,9 %) - ✓ Support prioritaire 24/7

    Trial¶

    Pour : évaluation de VOÏA avant engagement

    Limites : - 1 campagne - 2 utilisateurs maximum - Entreprises clientes illimitées - Durée : 1 mois

    Fonctionnalités : - ✓ Sondages classiques - ✓ Sondages conversationnels - ✓ Suivi NPS - ✓ Envoi par email - ✓ Exemples de modèles de sondage

    Comprendre les entreprises clientes¶

    Qu'est-ce qu'une entreprise cliente ?¶

    Une entreprise cliente est toute organisation dont les employés participent à vos sondages VOÏA, identifiée par le domaine de leur email (la partie après @).

    Par exemple, les participants alice@acme.com et bob@acme.com comptent tous deux comme une seule entreprise : acme.com.

    Ce qui n'est pas comptabilisé¶

    Les fournisseurs d'emails personnels (Gmail, Yahoo, Hotmail, Outlook, iCloud, etc.) sont exclus du comptage. Les participants utilisant ces adresses ne consomment jamais de créneaux entreprise.

    Comment le comptage fonctionne-t-il ?¶

    Le comptage est cumulatif à l'échelle du compte, toutes campagnes et toutes années confondues. Il n'est pas réinitialisé par campagne ni par année.

    Les entreprises déjà connues sont toujours autorisées¶

    Si le domaine d'une entreprise est déjà présent dans votre compte (au moins un participant avec ce domaine a déjà été ajouté), vous pouvez toujours ajouter de nouveaux participants de cette même entreprise, même si vous avez atteint votre limite.

    Cycle de licence¶

    Les licences fonctionnent sur un cycle annuel basé sur la date d'activation (pas le 1er janvier).

    Exemple : - Début de licence : 15 janvier 2025 - Renouvellement : 15 janvier 2026 - Réinitialisation du quota de campagnes : 15 janvier 2026

    Important : Le quota de campagnes se réinitialise à la date anniversaire. Le comptage d'entreprises clientes ne se réinitialise jamais — il est cumulatif sur toute la durée du compte.

    Dates de début et de fin personnalisées¶

    Les administrateurs de plateforme peuvent configurer une date de début et une date de fin personnalisées lors de l'attribution d'une licence — par exemple, pour antidater une licence afin de l'aligner sur la date de début d'un contrat, ou pour définir une expiration précise correspondant à un cycle de facturation. Contactez votre administrateur de compte si vous avez besoin d'une période de licence différente de la durée standard de 12 mois.

    Expiration de la licence¶

    30 jours avant l'expiration :
    Vous recevrez des rappels de renouvellement.

    À l'expiration : - Consultation des données existantes toujours possible - Impossible de créer de nouvelles campagnes - Impossible d'activer des campagnes en brouillon - Impossible d'ajouter des utilisateurs

    Renouvellement : Contactez votre administrateur de compte ou les ventes VOÏA.

    Suivi de consommation¶

    Accédez à Paramètres → Informations sur la licence pour consulter :

    • Campagnes utilisées cette année (ex. : « 3 sur 4 »)
    • Nombre d'utilisateurs actifs (ex. : « 7 sur 10 »)
    • Entreprises clientes chargées (ex. : « 14 sur 20 »)
    • Total de participants chargés dans VOÏA
    • Jours restants dans la période de licence
    • Date d'expiration

    Indicateurs de couleur : - Vert : Capacité disponible suffisante - Orange : Approche de la limite (75%+) - Rouge : Proche ou à la limite (90%+)

    Ce qui compte dans la consommation¶

    Campagnes : - Brouillons : ne comptent pas - Actives : comptent dans le quota - Terminées : comptent dans le quota (pour l'année)

    Utilisateurs : - Utilisateurs actifs : comptent - Utilisateurs désactivés : ne comptent pas - Invitations en attente : comptent

    Entreprises clientes : - Chaque domaine email d'entreprise distinct compte comme une entreprise - Les fournisseurs d'emails personnels (Gmail, Yahoo, Outlook, etc.) ne comptent jamais - Le comptage est cumulatif, toutes campagnes et toutes années confondues - Un domaine déjà connu du compte ne consomme jamais un crédit supplémentaire

    Participants : - Comptabilisés à titre informatif (total toutes campagnes) - Pas de blocage strict — un dépassement du seuil indicatif affiche uniquement un avertissement

    Mise à niveau ou rétrogradation¶

    Pack supplémentaire de campagnes¶

    Si vous avez besoin de campagnes supplémentaires au-delà de votre quota en cours d'année, un pack supplémentaire de campagnes est disponible. Chaque pack ajoute 4 campagnes supplémentaires à votre période de licence en cours, sans nécessiter de passage à une licence supérieure.

    Détails : - Disponible dans la période de licence en cours (non reporté à l'année suivante) - Accordé par un administrateur de plateforme — aucun achat en libre-service dans l'application - Tarification : Core — 8 500 $ / Plus — 15 000 $

    Pour demander un pack supplémentaire de campagnes, contactez votre administrateur de compte ou les ventes VOÏA.

    Quand mettre à niveau ?¶

    • Vous avez atteint votre quota de campagnes en cours d'année
    • Vous avez besoin de plus d'utilisateurs
    • Vous approchez de votre limite d'entreprises clientes
    • Vous souhaitez des fonctionnalités avancées (API, white-label)

    Processus : 1. Contactez votre administrateur de compte 2. Il contacte les ventes VOÏA 3. La mise à niveau est traitée sous 24 à 48 heures 4. Les nouvelles limites s'appliquent immédiatement

    Rétrogradation¶

    Les rétrogradations interviennent généralement au renouvellement.

    Restrictions importantes : - Si la rétrogradation réduit la limite d'utilisateurs en dessous du nombre actuel, vous devez désactiver des utilisateurs au préalable. - Si votre compte a déjà plus de 20 entreprises clientes chargées, une rétrogradation de Plus vers Core sera bloquée jusqu'à ce que vous soyez dans la limite Core.

    Application des limites¶

    Quota de campagnes atteint¶

    • Impossible d'activer de nouvelles campagnes
    • Les brouillons restent en brouillon
    • Les campagnes actives continuent normalement

    Quota d'utilisateurs atteint¶

    • Impossible d'inviter de nouveaux utilisateurs
    • Les utilisateurs existants continuent normalement

    Limite d'entreprises clientes atteinte¶

    Ce qui se passe : - Impossible d'ajouter des participants dont le domaine email est nouveau pour votre compte - Les participants issus d'entreprises déjà connues peuvent toujours être ajoutés - Les participants utilisant des adresses personnelles (Gmail, Yahoo, etc.) peuvent toujours être ajoutés - Les imports CSV qui dépasseraient la limite sont rejetés avec un rapport d'erreur listant chaque ligne concernée - Les administrateurs plateforme ne sont jamais bloqués

    Solutions : - Ajouter des participants issus d'entreprises déjà connues - Passer à Plus (100 entreprises) ou Pro (illimité) - Contacter les ventes VOÏA

    Volume de participants (seuil indicatif)¶

    Le nombre de participants par campagne est suivi à titre indicatif. Dépasser le seuil affiche un avertissement informatif, mais ne bloque pas l'ajout de participants.

    Types de licences spéciaux¶

    Comptes de démonstration¶

    • Usage : démonstrations et essais VOÏA
    • Très limités (1 campagne, 50 participants), courte durée
    • Non destinés à un usage en production
    • Pour convertir en licence complète : contacter les ventes VOÏA

    Licences Enterprise personnalisées¶

    Pour les grandes organisations ayant des besoins spécifiques : - Quotas de campagnes personnalisés (50+/an) - Limites d'utilisateurs personnalisées (100+) - Limites d'entreprises clientes personnalisées (ou illimitées) - Accords pluriannuels - Remises sur volume - Intégrations personnalisées

    Contactez les ventes VOÏA pour une licence personnalisée.

    Foire aux questions¶

    Puis-je dépasser temporairement mon quota de campagnes ?¶

    Non, le système applique strictement les limites de campagnes. Contactez les ventes pour une mise à niveau d'urgence.

    Que devient mon compte si ma licence expire ?¶

    Vos données sont conservées 90 jours après l'expiration. Vous pouvez vous connecter et consulter les données en lecture seule. Renouvelez dans les 90 jours pour éviter toute perte de données.

    Le comptage d'entreprises clientes se réinitialise-t-il chaque année ?¶

    Non. Le comptage est cumulatif et ne se réinitialise jamais. Il reflète tous les domaines distincts jamais ajoutés à votre compte.

    Puis-je ajouter des participants d'une entreprise déjà présente, même si j'ai atteint ma limite ?¶

    Oui. Dès qu'un domaine est connu de votre compte, vous pouvez toujours y ajouter des participants. Seuls les nouveaux domaines sont bloqués à la limite.

    Que se passe-t-il si un participant utilise une adresse Gmail ou Yahoo ?¶

    Les adresses email personnelles ne sont jamais comptabilisées comme une entreprise cliente et sont toujours autorisées, quelle que soit la limite.

    Quelle est la différence entre « entreprises clientes » et « participants » ?¶

    Un participant est un contact individuel. Une entreprise cliente est l'organisation pour laquelle il travaille, identifiée par son domaine email. Une entreprise peut avoir plusieurs participants.
