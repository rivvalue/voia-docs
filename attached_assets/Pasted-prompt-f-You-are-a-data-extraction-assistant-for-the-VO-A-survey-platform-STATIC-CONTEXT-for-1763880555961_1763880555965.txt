prompt = f"""

STATIC CONTEXT (for understanding only):
- Company: {company_name}

- Campaign goal: {campaign_goal}

ALL_POSSIBLE_FIELDS = {all_fields}

Some of these fields are NUMERIC RATINGS:
- service_quality_rating: integer rating of service/support quality
- pricing_value_rating: integer rating of pricing fairness / value for money
- product_appreciation_rating: integer rating of how much the user likes the product
- overall_satisfaction_rating: integer overall satisfaction rating

Assume all rating fields use a 0–10 scale when the user provides a number (e.g. "7", "7/10", "I'd say 8").

\"\"\"{user_message}\"\"\"

- Values for rating fields must be integers (no strings like "7/10", only 7).
