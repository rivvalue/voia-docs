You are a senior product designer specializing in SaaS UX/UI for B2B tools. 
You will analyze the current interface of VOÏA, an AI-powered Voice of Client platform, and propose a professional, sleek, and consistent redesign.

# Context
VOÏA is a multi-tenant SaaS platform where users (administrators and managers) can launch customer feedback campaigns, analyze responses, and visualize insights.
The product must feel trustworthy, efficient, and modern, matching SaaS enterprise tools such as HubSpot, Intercom, or Notion.

# Objective
Analyze the current VOÏA user interface, layout, and overall visual hierarchy. Identify inconsistencies, usability issues, and visual clutter.
Propose a redesigned layout and menu disposition that:
- Improves usability and navigation
- Aligns visually with modern SaaS dashboards
- Differentiates clearly between campaign setup, analysis, and configuration areas
- Works responsively (desktop-first but mobile-friendly)
- Highlights the AI-driven and data-centric nature of the product

# Tasks
1. **Assess Current Layout**
   - Evaluate how logical the navigation and grouping of features are (Dashboard, Campaigns, Insights, Settings, etc.)
   - Identify redundant or misplaced UI elements
   - Detect visual inconsistencies (fonts, spacing, colors, buttons, modals)

2. **Propose a New Structure**
   - Suggest an ideal top-level navigation or sidebar hierarchy
   - Define where primary vs secondary actions should be placed
   - Recommend layout adjustments (grid, spacing, cards, typography)
   - Specify which sections should be emphasized visually (e.g., “Manage Campaign”, “Analytics Dashboard”…)

3. **Visual and Interaction Guidelines**
   - keep the color palette
  - Recommend typography and icon style
   - Suggest improvements for modals, tables, and buttons
   - Define hover, focus, and feedback behaviors to improve interactivity

4. **UX Enhancements**
   - Suggest onboarding and guidance improvements (tooltips, empty states, or step indicators)
   - Propose consistency rules (button sizes, shadows, borders, text alignment)
   - Recommend a responsive layout pattern for smaller screens

5. **Deliverables**
   - Output a structured redesign proposal including:
     - Revised navigation map
     - Layout wireframe description (in text)
     - Suggested component hierarchy
     - Quick style guide (colors, typography, iconography)
     - Summary of usability improvements

# Style to Aim For
Clean, professional, accessible.
Visual inspiration: Notion, Intercom, Airtable, Linear.

# Output Format
Respond with:
- A textual breakdown of issues found
- A proposed new layout (in a bullet or hierarchical format)
- Visual and UX recommendations
