# Branding & personnalisation

---

## Logo

Ajoutez le logo de votre entreprise aux sondages et emails :

1. Aller dans **Configuration de la marque**
2. Cliquer sur **Téléverser un logo**
3. Sélectionner un fichier image (PNG, JPG ou SVG)
4. Le logo apparaît automatiquement sur l'ensemble de la plateforme

**Bonnes pratiques :**
- Utilisez un fond transparent (PNG)
- Taille recommandée : 200 x 60 pixels
- Taille maximale du fichier : 2 Mo
