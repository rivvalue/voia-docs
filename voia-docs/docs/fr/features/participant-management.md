# Gestion des participants

---

## Base centralisée

Base de données centralisée de tous vos clients :
- Nom et email
- Entreprise (optionnel)
- Rôle / poste (optionnel)
- Champs personnalisés

**Filtrage avancé :** Retrouvez des participants par entreprise, rôle ou attributs personnalisés

---

## Ajout de participants

**Option 1 : Saisie manuelle**
1. Cliquer sur **Ajouter un participant**
2. Saisir le nom, l'email et les informations optionnelles
3. Enregistrer

**Option 2 : Import CSV en masse** (recommandé pour les grandes listes)
1. Télécharger le modèle CSV
2. Remplir les données des participants
3. Téléverser le fichier
4. VOÏA valide et importe automatiquement

**Validation :** VOÏA vérifie les doublons d'email et les données invalides

---

## Segmentation

Regroupez les participants par :
- **Entreprise :** Suivez les retours par organisation
- **Rôle :** Dirigeants vs. utilisateurs terrain
- **Ancienneté :** Clients de longue date vs. nouveaux clients

**Utilisez la segmentation pour :**
- Des campagnes ciblées
- Des analyses segmentées
- Des expériences de sondage personnalisées
