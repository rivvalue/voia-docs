About Industry Topic Hints

How Industry Hints Work:

  • These hints customize AI prompts to ask more relevant, industry-specific questions
  • Each industry has topic-specific keywords that guide GPT-4o question generation
  • Business accounts can override specific hints via their custom configuration
  • Campaigns inherit hints from their business account or use platform defaults
  • Example: "Product Quality (focus on: defects, throughput, line reliability)"

Configuration (JSON Format)

Important: This editor modifies the industry_topic_hints_config.py file directly. Make sure your JSON is valid before saving. Changes apply immediately after save.

Nested JSON: industry name → topic name → comma-separated keywords.

{
  "YourIndustry": {
    "Product Quality": "keyword1, keyword2, keyword3",
    "Support Experience": "keyword4, keyword5, keyword6",
    "Service Rating": "keyword7, keyword8",
    "NPS Score": "keyword9, keyword10",
    "Pricing Value": "keyword11, keyword12",
    "User Experience": "keyword13, keyword14",
    "Satisfaction": "keyword15, keyword16",
    "Improvement Suggestions": "keyword17, keyword18"
  }
}

Edit in an external editor, then paste the full JSON here. Each industry key must match the account-level industry name exactly.

Each industry should have a dictionary of topics with comma-separated keywords
