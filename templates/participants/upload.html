Required CSV Format

Required columns:

  • email - Participant's email address
  • name - Participant's full name
  • company_name - Participant's company name

Optional segmentation columns:

  • role - Participant's role (e.g., "C-Level", "Manager")
  • region - Geographic region (e.g., "North America", "EMEA")
  • customer_tier - Customer segment (e.g., "Enterprise", "SMB")
  • language - Language code (e.g., "en", "es", "fr")
Example CSV with optional columns:
email,name,company_name,role,region,customer_tier,language
