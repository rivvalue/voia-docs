Required CSV Format

Required columns:

  • email - Participant's email address
  • name - Participant's full name
  • company_name - Participant's company name

Optional segmentation columns:

  • role - Participant's role (e.g., "C-Level", "Manager", "End User")
  • region - Geographic region (e.g., "North America", "EMEA", "APAC")
  • customer_tier - Customer segment (e.g., "T1: Strategic", "T2: Managed", "T3: Self-serve")
  • commercial_value - Company account value in USD (e.g., "250000"). All participants from the same company should have the same value
  • language - Language code (e.g., "en", "es", "fr")
Example CSV with optional columns:
email,name,company_name,role,region,customer_tier,language,commercial_value
